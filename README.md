# Proyecto-Final-POO-ElectroLab-Manager
Sistema Inteligente de Gestion de Componentes Electronicos
En los laboratorios de ingeniería electrónica y mecatrónica, la gestión de componentes constituye un aspecto crítico para el desarrollo de proyectos académicos y profesionales. La ausencia de un sistema integral que permita organizar inventarios, controlar préstamos y gestionar usuarios genera dificultades en la disponibilidad de materiales, retrasos en la ejecución de prácticas y riesgos de pérdida de recursos.  
El proyecto ElectroLab Manager surge como una solución informática orientada a optimizar la administración de componentes electrónicos en un entorno académico. Implementado en Java bajo los principios de la Programación Orientada a Objetos, el sistema busca proporcionar una plataforma confiable y eficiente que integre las siguientes funcionalidades:  
- Inventario de componentes: Registro, clasificación y consulta de resistencias, capacitores, transistores, microcontroladores y otros dispositivos, con validaciones de stock y alertas de disponibilidad.  
- Sistema de usuarios: Administración de perfiles con distintos niveles de acceso (estudiantes, docentes, administradores), garantizando seguridad y control en las operaciones.  
- Sistema de historial: Registro detallado de todas las operaciones realizadas (préstamos, devoluciones), permitiendo análisis del uso de los recursos.  
- Alertas automáticas: Notificaciones generadas por el sistema ante situaciones críticas, como bajo stock de componentes, retrasos en devoluciones o Componentes no disponibles.
- Gestión de préstamos: Registro de solicitudes de componentes, control de fechas de devolución y generación de reportes de uso.  
- Persistencia de datos: Almacenamiento mediante archivos o serialización de objetos, asegurando la conservación de la información entre sesiones.      
En síntesis, ElectroLab Manager se plantea como una herramienta académica y técnica que organiza los recursos del usuario.
